# 🎮 Projeto com Pygame

Este repositório contém um projeto simples desenvolvido em **Python** utilizando a biblioteca **Pygame**.

O objetivo do projeto é praticar conceitos de movimentação, colisão, renderização de texto e manipulação de eventos gráficos.

---

## 🚀 Sobre o Projeto

O programa cria uma janela gráfica onde dois textos se movimentam pela tela:

* **"TE FALEI CHINELÃO"**
* **"EU TE AMO GREMIO"**

Esses textos:

* Se movem continuamente pela tela
* Quicam nas bordas da janela
* Mudam de cor ao bater nas paredes
* Colidem entre si e também mudam de cor

O projeto simula um pequeno sistema de colisão utilizando `Rect` do Pygame.

---

## 🧠 Conceitos Utilizados

* Inicialização de janela gráfica
* Loop principal do jogo
* Controle de FPS
* Renderização de texto
* Movimentação com velocidade em X e Y
* Detecção de colisão com bordas
* Detecção de colisão entre objetos
* Geração de cores aleatórias

---

* Python 3
* Biblioteca Pygame
